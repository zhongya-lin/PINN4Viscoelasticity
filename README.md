# Physics-informed neural network for viscoelastic deformation, buckling and tissue growth

This repository provides numerical examples of physics-informed neural network-based computational viscoelasticity framework. This study aim to simulate the viscoelastic buckling, tissue growth and morphological evolution. 

## Environment

- Python 3.10
- PyTorch 2.4.1
- Numpy 2.1.1

## Paper link

- [A physics-informed neural network framework for simulating creep buckling in growing viscoelastic biological tissues](https://doi.org/10.1016/j.cma.2025.118715), Computer Methods in Applied Mechanics and Engineering, 2026, 452-A, 118715.

- [A physics-informed neural network framework for simulating creep buckling in growing viscoelastic biological tissues](https://arxiv.org/abs/2506.18565), arXiv, 2025.
